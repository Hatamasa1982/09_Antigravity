# 株式会社H・O・C 面接振り返り・学び（2026/04/01）

## 1. 面接の概要
- **日程**: 2026年4月1日
- **ポジション**: IT営業（実態はOA機器販売）
- **選考状況**: 一次面接（現場マネージャー・実務担当者）

## 2. 衝撃！現場のリアル（一次情報）
現場を見せてもらい、現代のIT・DXのイメージとは真逆の、非常に強度の高い営業スタイルを体感した。

### 🚀 営業の「質」と「量」
- **テレアポ**: 1日 **100〜200件**。
- **歩留まり**: 100件中、話せるのは20件。アポまで至るのはわずか **1〜2件**。
- **商材**: シャープ製コピー機（1台 約150万円）。
- **短期決戦**: 出会って **2時間で即決** させる極めて高い営業力が求められる。
- **リプレイス**: 平均3年周期での切り替えサイクル。

### 💰 評価・ビジネスモデル
- **完全成果主義**: 売り上げれば給料・役職が上がる。入社1年での管理職も可能。
- **評価基準**: ノルマはないが、赤字を出さないことが絶対条件。利益の一定数がインセンティブとなる。
- **競合他社比較**:
    - リコー・ゼロックス：ブランド力はあるが、価格が高すぎる。
    - シャープ：安価で性能も遜色ない。価格競争力で勝負する営業スタイル。

## 3. 面接官からの問いかけ（想定と実際）
- **深掘り**: 「これほど素晴らしい経歴で、なぜうちなのか？」「なぜ前職を辞めたのか？」と、ミスマッチへの警戒が強かった。
- **覚悟の確認**: 40代という年齢を踏まえ、「すぐ辞めてはお互いのためにならない」と、包み隠さず過酷な現場を見せてくれた点は誠実だった。
- **その他**: 家族構成や、社内での具体的なDX実装経験についても質問があった。

## 4. 自身の判断・収穫
### ⚖️ 最終判断
- **「絶対に行かない」**:
    - 20年間のキャリアと、提供したい「IT×DXの価値」が、1日200件のテレアポという手段と大きく乖離している。
    - 生活が極限まで困窮しない限り、自ら進んで選ぶ環境ではないと確信。

### 💡 学び（ネタとしての価値）
- **営業力の再考**: 短時間で高額な意思決定をさせる「瞬発力」を目の当たりにし、営業の幅広さを学んだ。
- **価値観の明確化**: 自分が大切にしたい「長期的信頼関係」と「現場の仕組み化」の重要性を再認識できた。
- **最高のビジネスネタ**: 「令和の時代に残る衝撃的な営業現場」として、今後のコミュニケーションの武器にする。
