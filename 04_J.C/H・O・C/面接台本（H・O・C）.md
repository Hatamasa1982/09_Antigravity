# 株式会社H・O・C 面接台本（ITサポート営業）

## 1. 志望動機（業務内容を核とした5段構成）
**① 結論**
「貴社のITサポート営業として、地域企業が抱える『IT導入の壁』を取り払い、お客様が本業に集中できる環境作りに【貢献】したいからです。」

**② 前提**
「私は20年間の旅行業において、担当エリアへ足しげく通う泥臭い対面営業を通して、経営者の方々との深い信頼関係を築いてきました。」

**③ 本題**
「その一方で、数多くの中小企業がIT化の遅れや人手不足に悩み、本来の業務効率を落としている実態にも直面してきました。私自身、現場の非効率をGASやAppSheetを用いて自力で改善し、生産性を15倍に高めた経験があります。この経験から、ただモノを売るのではなく、『IT×人』の力で地域企業の業務を根本から底上げしたいと考えるに至りました。」

**④ 補足**
「その中で、対面でのコミュニケーションや『人と人とのつながり』を重んじ、B.LEAGUE等を通じた地方創生にも本気で取り組む貴社の『社会性と事業性を追求する姿勢』に強く共感しています。」

**⑤ まとめ**
「私が20年で培った『泥臭い関係構築力』と、『自律的に学び実務へ落とし込むDX実装力』を掛け合わせ、いちばん身近なパートナーとして地域企業のDX化推進と貴社の業績拡大に貢献します。」

---

## 2. 転職理由（ポジティブ・トランスフォーメーション）
「現職でも成果を出し、人間関係にも恵まれておりますが、私自身の『生産性を最大化し、より多くの企業に本質的な価値を届けたい』という成長欲求を実現するために転職を決意しました。
旅行業というアナログな環境において、私は独学でプログラミング（GAS等）を学び、15人時かかっていた業務を1時間に短縮するなどの仕組み化を実現しました。しかし、現職の枠組みではあくまで『自社内の局所的な改善』に留まります。
この身につけたITスキルと、長年培った営業力をフルに活かし、『地域企業全体のDX支援』という、より視座の高いフィールドでの挑戦がしたいと考えています。」

---

## 3. 自己PR（STAR法：強みと企業のニーズの合致）
**■ 強み：「現場起点でのDX実装力と、泥臭い関係構築力」**
- **状況(S)**: 地方エリア担当時、訪問頻度が限られる中で売上向上が課題でした。また社内ではアナログな集客管理が常態化し、多大な工数を要していました。
- **課題(T)**: 顧客との接触の質を高めつつ、社内業務を極限まで効率化する必要がありました。
- **行動(A)**: 営業面では単なる御用聞きではなく、地域データの集計レポートなど「相手のタメになる情報価値」の提供に徹しました。同時に、社内業務では独学でGASとGoogleフォームを連携させ、独自の管理ツールを開発しました。
- **結果(R)**: 結果として、担当エリアの売上を3年で1.5倍に拡大させながら、事務作業の工数を1/15へと激減させました。
- **マッチング**: 貴社においても、まずはこの「顧客に寄り添う泥臭い営業力」で確固たる信頼を獲得し、その上で「現場目線のIT改善提案」を行うことで、お客様から真に『ありがとう』と言われる営業ができると確信しています。20代・30代の方が多い組織におきましても、年間500時間以上の継続学習の習慣を活かし、誰よりも早くキャッチアップいたします。

## 4. 弱みの昇華
**■ 弱み：「人間の記憶力への依存や、細かなケアレスミス」**
「私の弱みは、かつて人間の記憶力に頼ったタスク管理をしており、細かなミスが生じていた点です。しかし、この数年間で『人間がやらなくていいことは全てITに任せる』という仕組み化を徹底しました。GASでの自動リマインドやデータベースでの一元管理を自作し生活に導入した結果、ミスが撲滅されたことはもちろん、空いたリソースを『顧客との対面での深いコミュニケーション』という、人間にしかできない価値創造に全振りできるようになりました。」
