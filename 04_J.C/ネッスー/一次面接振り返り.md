# ネッスー株式会社：一次面接 振り返りメモ（質問集）

一次面接で実際に聞かれた質問事項をまとめました。
今後の最終面接対策や、他社の選考時の自己分析のブラッシュアップに活用できる貴重な資産です。

## 1. 基本的な質問
- 志望動機
- カジュアル面談の感想
- 直近1〜2年の業務履歴

## 2. 実務・課題解決に関する質問（再現性の確認）
- 数字に厳しい状況の対応は？（例：物価高を価格転嫁できるか？）
- これまでに一番ヘビーだった仕事や交渉事は？また、それをどう解決したか？
- マーケティングの分野で何ができるか？
- 配送（物流・オペレーション）の部分で何ができるか？

## 3. キャリアと志向性に関する質問（0→1フェーズへの適性）
- 「0→1」と「1→10」はどちらが得意か？
- 何か「0→1」にした経験はあるか？
- 3〜5年後のビジョンは？

## 4. チームワーク・マネジメントに関する質問
- チームで働くときに、自分はどんな価値を出せるか？
- 部下が成果を出せない場合、どのようにマネジメントするか？

## 5. 自己認識・他己評価に関する質問
- 自身の短所は？
- 自身の長所と短所について、上司からどんなフィードバックがあったか？
