# 隣ラジオ コメント録

ここは、Podcast「隣ラジオ」への毎日のコメントを資産として蓄積していくための専用フォルダです。

## 使い方

1. あなたは毎日、私（Antigravity）へのチャットで **「今日の隣ラジオのコメントはこれです：〇〇〇」** と送るだけです。
2. 私がそれを受け取り、自動的にこのフォルダ内に新しいテキストファイル（Markdown形式）を作成し、日付をつけて保存します。

## ここに貯まったコメントの活用方法

- 面白かった回や考えさせられた回を後から振り返る日記として。
- noteの記事の「構成案」やSNS（XやInstagram）の「投稿の種」として。

面倒な整理は私に任せて、あなたの思考をどんどんアウトプットしてくださいね！
