# 2日目：焼肉のタレで味付け一発！野菜かくれんぼのスタミナ焼肉丼

味付けは市販の焼肉のタレだけで完了！細かく刻んだ野菜をお肉と一緒に甘辛く炒めてごはんに乗せるだけです。

## 1. 材料リスト（4人分）
- **ごはん:** 3合（炊きたて）
- **豚バラ肉（または牛薄切り肉）:** 400g
- **玉ねぎ:** 1/2個
- **ピーマン:** 2個
- **キャベツ:** 2〜3枚
- **サラダ油:** 大さじ1
- **ごま油:** 小さじ1（仕上げ用）

**【味付け】**
- **市販の焼肉のタレ（甘口〜中辛）:** 大さじ5〜6（お好みで調整）

## 2. 手順
1. **お肉を切る:** 豚バラ肉は食べやすい大きさに切る。
2. **野菜を細かく刻む:** 玉ねぎ、ピーマン、キャベツを細かいみじん切りにする。
3. **野菜を炒める:** フライパンにサラダ油を熱し、みじん切りにした野菜を中火でしっかり炒める。（カサが減ってしんなりするまで）
4. **お肉を炒める:** フライパンの端に野菜を寄せ、空いたところでお肉を炒める。お肉の色が変わったら全体を混ぜ合わせる。
5. **味付け:** 焼肉のタレを回し入れ、汁気が少なくなるまで強火でサッと炒め合わせる。風味付けにごま油を垂らす。
6. **盛り付け:** 丼にごはんを盛り、フライパンの中身をたっぷり乗せたら完成！

## 3. キッズ・ポイント
- **野菜を隠す工夫:** ピーマンやキャベツも細かく刻んでしまうことで、焼肉のタレの濃い味にコーティングされ、全く気にならなくなります。
- **子供が喜ぶ味付け:** 子供が大好きな焼肉のタレを使うことで、野菜への抵抗感をゼロに。豚バラ肉の脂の旨みで、ごはん3合があっという間になくなるがっつり系です。小1のお子様に合わせてタレは甘口を選ぶのがおすすめです。
