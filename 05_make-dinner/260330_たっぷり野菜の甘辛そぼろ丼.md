# [本日 3/30] [ワンパン] たっぷり野菜の甘辛そぼろ丼

## 1. 材料リスト（4人分）
- 豚ひき肉（または合い挽き肉）：400g
- ピーマン：2〜3個
- ごま油：大さじ1

**[A] 合わせ調味料**
- 醤油：大さじ4
- みりん：大さじ4
- 砂糖：大さじ3
- 酒：大さじ2

## 2. 彩り小鉢（きゅうりのツナ塩昆布和え）
- きゅうり：1〜2本
- ツナ缶：1缶
- 塩昆布：大さじ1
- ごま油：小さじ1
**手順：** きゅうりを薄切りにし、油を切ったツナ、塩昆布、ごま油とボウルで和えるだけ。（火を使わない1分小鉢！）

## 3. 手順
1. **野菜を切る**：ピーマンの種とヘタを取り、できるだけ細かくみじん切りにします。
2. **炒める**：フライパンにごま油を熱し、ひき肉とみじん切りのピーマンを中火で炒めます。
3. **味付け**：肉の色が変わったら、[A]の調味料をすべて加え、フライパンの底にごくわずかに汁気が残る程度までしっかり煮詰めます。
4. **仕上げ**：3合炊いたご飯にたっぷりかけて完成です！

## 4. キッズ・ポイント
- ピーマンは限界まで細かく刻むことで、お肉と甘辛ダレに完全に同化します！「今日のお肉おいしいね！」と食べてくれるはずです。
- お好みで真ん中に温泉卵や炒り卵を乗せると、黄色が追加されて彩りもアップし、お子様のテンションもさらに上がります。
