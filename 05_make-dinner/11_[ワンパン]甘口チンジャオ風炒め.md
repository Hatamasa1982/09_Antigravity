# [明日 3/31] [ワンパン] 豚肉とピーマンの甘口チンジャオ風炒め

## 1. 材料リスト（4人分）
- 豚肉（切り落としなど）：400g
- ピーマン：3〜4個
- 片栗粉：大さじ1（豚肉にまぶす用）
- ごま油：大さじ1

**[B] 甘口タレ**
- オイスターソース：大さじ2
- 醤油：大さじ1
- 砂糖：大さじ1
- 鶏ガラスープの素：小さじ1
- 水：大さじ2

## 2. 彩り小鉢（大根とカニカマのさっぱり和え）
- 大根：1/4本（約10cm）
- カニカマ：1パック
- **味付け**：ポン酢（大さじ1）、ごま油（小さじ1）、砂糖（ひとつまみ）
**手順：** 大根を細い千切りにし、手でさいたカニカマと調味料で和えるだけ。（火を使わない1分小鉢！）

## 3. 手順
1. **下準備**：豚肉は食べやすい大きさに切り、ポリ袋に入れて片栗粉をまぶして揉み込みます。ピーマンは極細切りにします。
2. **調味料を混ぜる**：[B]の調味料をすべて小さな器で混ぜ合わせておきます。
3. **炒める**：フライパンにごま油を中火で熱し、豚肉を炒めます。肉に火が通ったらピーマンを加えてサッと炒めます。
4. **仕上げ**：混ぜておいた[B]のタレをフライパンに回し入れ、全体にとろみがつき、タレがよく絡んだら完成です！

## 4. キッズ・ポイント
- ピーマンは極細切りにすることで苦味を感じにくくなり、お肉にまぶした片栗粉のおかげで甘口のタレがしっかりと絡むのでとても食べやすくなります。
- 副菜の和え物は、カニカマの甘みとごま油の香りで、酸っぱいものが苦手なお子様でも食べやすい味付けです。マヨネーズを使わないのでさっぱりと頂けます。
