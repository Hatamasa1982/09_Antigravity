# 3日目：火を使わない！お刺身パックの二色丼 ＆ レンジで即席・野菜とろける豚汁

大好物のお刺身をメインに！副菜代わりに、レパートリーにある「豚汁」をレンジで超時短で作るセットです。

## 1. 材料リスト（4人分）
**【お刺身二色丼】**
- **ごはん:** 3合（少し冷ました状態、または酢飯にしてもOK）
- **サーモンの刺身:** 2パック（柵でもスライスでも）
- **マグロの刺身:** 2パック
- **醤油・わさび:** 適量（子供はわさび抜きで醤油のみ）

**【即席レンジ豚汁】**
- **豚こま切れ肉:** 150g
- **大根:** 3cm分
- **にんじん:** 1/3本
- **水:** 600ml
- **和風だしの素（顆粒）:** 大さじ1
- **味噌:** 大さじ3〜4

## 2. 手順
1. **【豚汁の準備】野菜を「すりおろす」か「極細切り」に:**
   大根とにんじんを、すりおろし器ですりおろすか、スライサーで極細切りにする。（これが火の通りを究極に早くし、野菜の食感を消すコツです！）
2. **【豚汁】レンジで加熱:**
   大きめの耐熱ボウルに、豚肉、すりおろした野菜、だしの素、水（半量の300ml）を入れる。ふんわりラップをして電子レンジ（600W）で約6〜7分加熱し、お肉にしっかり火を通す。
3. **【丼】お刺身の準備:**
   レンジで加熱している間に、お刺身をごはんの上に乗せて「お刺身二色丼」を完成させる。
4. **【豚汁】仕上げ:**
   レンジからボウルを取り出し（やけど注意）、残りのお湯（または水 300ml）を加え、味噌を溶かし入れる。
5. **盛り付け:**
   お刺身丼と、お椀によそった豚汁を並べて完成！

## 3. キッズ・ポイント
- **野菜を隠す工夫:** 豚汁の野菜（大根・にんじん）を「すりおろす」ことで、スープに完全に溶け込みポタージュのようになります。野菜嫌いな子でも汁ごと飲んで栄養補給ができます！
- **子供が喜ぶ味付け:** メインはお子様の大好きな「お刺身」なので満足度100%！火を使わずに完了するので、お父さんの負担も最小限です。
