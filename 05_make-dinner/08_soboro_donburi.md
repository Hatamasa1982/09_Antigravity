# 野菜たっぷり甘辛そぼろ丼

パパにも作りやすい、フライパン1つで完成するワンパンメニュー。野菜を細かく刻んで隠すことで、お子様もモリモリ食べてくれます！

## 1. 材料リスト（4人分）
- **ごはん:** 3合（炊きたて）
- **豚ひき肉（または合い挽き肉）:** 400g
- **玉ねぎ:** 1個
- **にんじん:** 1/2本
- **ピーマン:** 2個
- **トッピング用卵:** 4個（※市販の温泉卵がおすすめ）
- **サラダ油:** 大さじ1

**【合わせ調味料】**
- **醤油:** 大さじ4
- **砂糖:** 大さじ3
- **みりん:** 大さじ2
- **酒:** 大さじ2
- **すりおろしニンニク（チューブ可）:** 小さじ1
- **すりおろし生姜（チューブ可）:** 小さじ1

## 2. 手順
1. **野菜を細かく刻む:**
   玉ねぎ、にんじん、ピーマンを「細かいみじん切り」にする。（野菜カッターやフードプロセッサーがあれば一瞬です）
2. **調味料を混ぜておく:**
   【合わせ調味料】の材料をすべて小さめの器に入れ、混ぜ合わせておく。
3. **野菜を炒める:**
   フライパンにサラダ油をひいて中火で熱し、刻んだ野菜を入れてしんなり透き通るまで炒める。
4. **ひき肉を炒める:**
   野菜が柔らかくなったらひき肉を加え、肉の色が完全に白っぽく変わるまでほぐしながら炒める。
5. **味付けをして煮詰める:**
   あらかじめ混ぜておいた【合わせ調味料】をフライパンに流し入れる。全体をかき混ぜながら、汁気が少しとろっと少なくなるまで3〜4分ほど煮込む。
6. **卵の準備:**
   別のフライパンで目玉焼きやスクランブルエッグを作る。（市販の温泉卵ならこの工程はスキップ）
7. **盛り付け:**
   ごはん3合を4つの丼に分け、そぼろをたっぷりのせ、最後に卵をトッピングして完成！

## 3. キッズ・ポイント
- **野菜を隠す工夫:**
  野菜をとにかく「細かく刻んで」お肉と一緒に煮込む！お肉の旨みと甘辛だれが染み込み、野菜嫌いのお子様でもお肉感覚でパクパクいけます。
- **子供が喜ぶ味付け・盛り付け:**
  トッピングの卵の黄身を崩すと味がまろやかになり、小1のお子様も食べやすくなります。小5のお子様には、熱いうちに「とろけるチーズ」を乗せるとボリュームと満足度がアップします！
