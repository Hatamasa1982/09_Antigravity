# [ワンパン・野菜隠し] 鶏もも照り焼きつくね

**作成日**: 2026年4月2日

## 1. 家族構成とボリューム
- **人数:** 大人2名、小5、小1（計4名）
- **主食:** お米3合にぴったりの味付けとボリューム！
- **特徴:** マヨネーズ不使用。豆腐をつなぎに使ったふわふわ食感。

## 2. 材料リスト (4人分)
### メイン（肉だね）
- **鶏ひき肉（または細かく刻んだ鶏もも肉）**: 400g
- **玉ねぎ**: 2/3個（または小さめ1個）
- **人参**: 1/3本
- **ピーマン**: 2個
- **絹ごし豆腐**: 100g（水気を軽く切る）
- **片栗粉**: 大さじ2
- **塩コショウ**: 少々

### 照り焼きのタレ
- **醤油**: 大さじ2
- **酒**: 大さじ2
- **みりん**: 大さじ2
- **砂糖**: 大さじ1

## 3. 調理手順 (20分以内)
1. **野菜を隠す準備**
   - 玉ねぎ、人参、ピーマンを**すべて細かいみじん切り**にする（チョッパー推奨）。
   - みじん切りにした野菜を耐熱容器に入れ、ふんわりとラップをして電子レンジ（600W）で2〜3分加熱し、粗熱を取ります。これで野菜が柔らかく甘みも増します！
   - **キッズ・ポイント**: 野菜が苦手な子のために、できるだけ細かくして肉に馴染むようにします。
2. **肉だねを混ぜる**
   - ボウルに肉、野菜、豆腐、片栗粉、塩コショウを入れ、粘りが出るまで手でよく練り混ぜる。
3. **形を整えて焼く**
   - フライパンに油（分量外）を中火で熱し、一口大の小判形にして並べる。
   - 焼き色がついたら裏返し、蓋をして弱火で4〜5分、中まで火を通す（蒸し焼き）。
4. **タレを絡める**
   - 混ぜ合わせた【照り焼きのタレ】を回し入れ、強火でタレにトロミが出て照りが出るまで煮詰めながら全体に絡める。

## 4. 彩り小鉢 (1分完成)
**「やみつき叩ききゅうり」**
- きゅうり（1本）を袋に入れて棒などで叩いて割り、塩昆布（少々）とごま油（適量）を入れて揉むだけ。

## 5. 盛り付け・保存のコツ
- 彩りにミニトマトを添えると完璧です。
- 翌日のお弁当のおかずにも最適です。
