# [明後日 4/1] [ワンパン] 小葱と豚バラの塩だれ炒め

## 1. 材料リスト（4人分）
- 豚バラ肉：400g
- 小葱：1束（たっぷり全て）
- ごま油：大さじ1

**[C] 特製塩だれ**
- 鶏ガラスープの素：大さじ1
- 塩：小さじ1/2
- レモン汁（またはお酢）：小さじ1
- にんにくチューブ：約2cm
- 黒こしょう：少々（小1のお子様が気になる場合は省いてOK）

## 2. 彩り小鉢（叩ききゅうりのごま油和え）
- きゅうり：2本
- ごま油：大さじ1
- 鶏ガラスープの素：小さじ1
- 塩：ひとつまみ
- 白ごま：適量
**手順：** きゅうりを麺棒などで叩いて割り、手で一口大にちぎります。調味料すべてと一緒にポリ袋に入れて揉み込みます。（火を使わない1分小鉢！）

## 3. 手順
1. **切る**：豚バラ肉はひと口大に切ります。小葱は根元を切り落とし、3〜4cmの長さに揃えて切ります。
2. **調味料を混ぜる**：[C]の特製塩だれを全て器に混ぜておきます。
3. **炒める**：フライパンにごま油を中火で熱し、豚バラ肉をカリッとするまで炒めます。（脂がたくさん出る場合はキッチンペーパーで軽く拭き取ります）
4. **仕上げ**：小葱を加え、サッと火を通したら、混ぜておいた[C]の塩だれを入れて全体にサッと絡めて完成です！

## 4. キッズ・ポイント
- 豚肉の脂のコクと旨味、さらにレモンの効いたさっぱり塩だれが食欲をそそるので、小葱も一緒に違和感なく美味しく食べられます。
- きゅうりは叩いて割ることで、塩気とごま油の風味が即座に染みこみ、スナック感覚でポリポリ食べられます！
