# X (旧Twitter)コンテンツ生成スキル

このスキルは、X (旧Twitter)に最適化されたコンテンツを生成するためのものです。
AIアシスタントは、このスキルを読み込み、X (旧Twitter)のコンテンツ生成において、このスキルに沿った内容を作成してください。

---

## 1. 基本設定

### 文字数制限
- 推奨文字数: 100〜200文字(読みやすさ重視)
- 最大文字数: 280文字(日本語の場合、全角140文字程度)

### 推奨構成
【フック(1行)】
問いかけや共感で注意を引く
【本文(2〜3行)】
具体的な内容や気づき
【CTA(1行)】
行動喚起

### 改行ルール
- 1〜2行ごとに改行して読みやすくする
- 空行は使わない（タイムラインで空行が詰まるため）

### 絵文字使用ルール
控えめに使う(1〜2個程度)。フックやCTA部分に配置。

---

## 2. プラットフォーム固有の設定

### ハッシュタグ戦略
- 使用しない（リーチを伸ばすため）

### エンゲージメント最適化
- 問いかけを積極的に使い、返信や引用RTを促進する
- 引用RT促進：「あなたはどう思いますか?」など
- リプライ促進：「コメントで教えてください」など

---

## 3. CTA(行動喚起)パターン

### 優先順位
1. リプライ促進(エンゲージメント重視)
2. プロフィールへ誘導(フォロー促進)
3. 引用RT促進(拡散重視)
4. リスト獲得(外部リンク)

### 具体例
「みんなの会社のFAX事情もコメントで教えてください！」
「プロフからすぐ使える自動化テンプレを配布中👇」

---

## 4. AI生成時の注意事項

### 必ず守ること
- ビジネス上流設計(`.agents/rules/mybusiness.md`)のトーン&マナーを遵守すること
- X (旧Twitter)固有の制限やルールを守ること
- **話し言葉**で、息継ぎや間が伝わるように作成すること

### 推奨すること
- 読者への問いかけを適宜入れ、エンゲージメントを高める
- ベネフィットを具体的に提示する
- 自身の体験やストーリーを交えて信頼構築を行う

### 避けること
- 専門用語の多用(ターゲットが理解できない言葉)
- 根性論や具体性のないアドバイス
- 冗長な導入や結論の見えない構成
