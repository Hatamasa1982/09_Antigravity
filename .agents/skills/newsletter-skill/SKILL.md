# メルマガコンテンツ生成スキル

このスキルは、メルマガに最適化されたコンテンツを生成するためのものです。
AIアシスタントは、このスキルを読み込み、メルマガのコンテンツ生成において、このスキルに沿った内容を作成してください。

---

## 1. 基本設定

### 文字数制限
- 推奨文字数: 800〜1,500文字(読みやすさと情報量のバランス)
- 最小文字数: 500文字

### 推奨構成
【件名】
開封率を高める魅力的な件名
【挨拶(1〜2行)】
親しみやすい挨拶
【フック(2〜3行)】
問題提起や共感
【本文(10〜15行)】
具体的な内容、ストーリー、ノウハウ
【まとめ(2〜3行)】
要点の整理
【CTA(2〜3行)】
行動喚起
【署名】
名前、プロフィールリンク

### 改行ルール
1文書いたら改行するなど、スマホで読みやすいように空行を適度に配置する

### 絵文字使用ルール
適度に使用し、親しみやすさを演出する

---

## 2. プラットフォーム固有の設定

### 件名の最適化
- 文字数: 15〜25文字(スマホで全文表示)
- パターン: 問いかけ型(「〜していませんか?」)、数字型(「3つの〜」)、緊急性型(「今すぐ〜」)、共感型(「〜ですよね」)

### 教育フェーズの設定
- **フェーズ1: 信頼構築(1〜3通目)**: 自己紹介、共感、価値提供(ストーリー、問題提起、小さなノウハウ)
- **フェーズ2: 教育(4〜7通目)**: 独自アプローチの理解、価値の認識(他の選択肢の評価、独自アプローチ)
- **フェーズ3: セールス(8通目以降)**: オファーの提示、購入促進(オファーの価値、未来のビジョン)

### 配信頻度
- 通常: 週1〜2回
- ステップメール: 毎日〜2日に1回

---

## 3. CTA(行動喚起)パターン

### 優先順位
1. リンククリック(LP、記事、動画など動画への誘導)
2. 返信促進(エンゲージメント向上)
3. SNSフォロー促進
4. 商品購入(セールスフェーズ時)

### 具体例
「今日のメールを読んでどう思いましたか？ぜひこのメールにそのまま返信して教えてください！」
「詳しい解説文やテンプレートはこちらのリンクから動画をご覧ください。」

---

## 4. AI生成時の注意事項

### 必ず守ること
- ビジネス上流設計(`.agents/rules/mybusiness.md`)のトーン&マナーを遵守すること
- メルマガ固有の制限やルールを守ること
- **話し言葉**で、息継ぎや間が伝わるように作成すること

### 推奨すること
- 読者への問いかけを適宜入れ、エンゲージメントを高める
- ベネフィットを具体的に提示する
- 自身の体験やストーリーを交えて信頼構築を行う

### 避けること
- 専門用語の多用(ターゲットが理解できない言葉)
- 根性論や具体性のないアドバイス
- 冗長な導入や結論の見えない構成
