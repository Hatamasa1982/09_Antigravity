単にITツールを導入するのではなく、現場の痛みを特定し、最小のリソースで最大の生産性向上を実現する実務能力です。

* **圧倒的な業務自動化の実績と再現性**
    * [cite_start]**実績**: 1,000名規模の集客管理において、従来5名が半日（計20人時）を費やしていたアナログ作業を、独学で習得したGoogle Apps Script (GAS) を用いて自動化 [cite: 7, 20][cite_start]。ボタン一つ、1名、1時間（1人時）で完結させる仕組みを構築した [cite: 7, 20]。
    * [cite_start]**価値**: 既存のGoogle Workspace環境を徹底活用し、一人当たりの生産性をコロナ前比で1.75倍まで高めた実績を持つ [cite: 18, 19]。
* **生成AI・ITツールの実務適用能力**
    * [cite_start]**実績**: 生成AI（文章・画像・動画生成）を駆使し、修学旅行提案用の30秒オープニングムービーを自作 [cite: 9, 19][cite_start]。視覚的インパクトによる差別化で、無名状態だった教育旅行分野において新規2件の受注に成功した [cite: 20]。
    * [cite_start]**アセット**: AppSheetを用いた簡易業務アプリの作成、Slack/Notionによるナレッジ管理を実務で運用 [cite: 19, 20][cite_start]。技術を「意思決定を後押しする手段」として使いこなす [cite: 20]。
* **組織横断型DX推進とコミュニティ運営**
    * [cite_start]**実績**: 地区や役職の壁を越え、社内初となる「AI活用・業務改善コミュニティ」をゼロから立ち上げ、全国30名規模の学習基盤へ発展させた [cite: 5, 11, 20]。
    * [cite_start]**価値**: 自身の知見を惜しみなく発信し、アナログな環境でも周囲を巻き込んで組織全体のデジタルリテラシーを底上げする「DXエバンジェリスト」として機能する [cite: 8, 20]。

## 2. 逆境を突破する戦略的セールス・関係構築スキル
43歳のキャリアで培った、顧客の課題を定義し、信頼を数字に変えるプロフェッショナルな営業力です。

* **付加価値提供による市場開拓（道南地区の成功事例）**
    * [cite_start]**状況**: 札幌から片道4時間、地元志向が極めて強い競合優位のエリアを担当 [cite: 7, 21]。
    * [cite_start]**行動**: 訪問頻度で勝負せず、ホテルの混雑予測やイベント情報を集約した「会えば必ず得るものがある」情報価値の提供を徹底 [cite: 7, 21]。
    * [cite_start]**結果**: 顧客の期待値を高め、3年間で売上を平均6,000万円から9,000万円（1.5倍）へ拡大させた [cite: 8, 21]。
* **究極の信頼資本とステークホルダー調整力**
    * [cite_start]**証左**: 顧客から「あなたに無理と言われたら納得（諦め）がつく」と言わしめるほどの誠実な対応力を持つ [cite: 12]。
    * [cite_start]**実績**: 添乗中のトラブル（生徒の遅延等）において、航空会社等と粘り強く調整を行い、通常不可とされる対応を引き出し全員での出発を実現させた [cite: 21]。

## 3. 心理的安全性を重視したマネジメントと組織貢献
ミドル層として期待される、組織を維持し次世代を育成するリーダーシップです。

* **若手育成と定着率の維持**
    * [cite_start]**実績**: コロナ禍後のリソース不足時、負担過多による離職危機に対し、教育体制を「1年間の徹底的な伴走と心理的サポート」へ抜本的に見直し [cite: 12, 21]。
    * [cite_start]**結果**: 厳しい環境下でも2022年以降の若手定着率を維持した [cite: 21]。
* **正解のない状況下での意思決定（決断経験）**
    * [cite_start]**実績**: 社員数が1/4（60名→15名）に激減したコロナ禍において、YouTubeでの物販、ワクチン接種受付、スーパーでの品出しなど、既存の枠に捉われない新規領域への挑戦をマネジメント [cite: 7, 14]。
    * [cite_start]**バリュー**: 「リスクは7割の成功可能性がある時にとる」という実戦的な判断基準と、激変期を生き抜いたレジリエンスを持つ [cite: 14]。

常にアップデートを止めない姿勢と、ITを駆使した徹底的な自己管理能力です。

* **圧倒的な学習意欲と継続力**
    * [cite_start]**習慣**: 毎朝5:00〜6:30の90分を学習に充て、年間500時間以上の自己研鑽（AI活用、プログラミング等）を4〜5年継続している [cite: 7]。

* **「弱み」を「ITの仕組み」で解決する自己管理**

    * [cite_start]**具体策**: Excel関数による自動チェック、プログラミングによる通知、メモのデジタル化と検索・リマインド機能の活用により、ミスを未然に防ぐ高度な自己管理体制を構築している [cite: 9, 10]。
