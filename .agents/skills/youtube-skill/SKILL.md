# YouTube台本コンテンツ生成スキル

このスキルは、YouTube台本に最適化されたコンテンツを生成するためのものです。
AIアシスタントは、このスキルを読み込み、YouTube台本のコンテンツ生成において、このスキルに沿った内容を作成してください。

---

## 1. 基本設定

### 文字数制限
特になし（動画の尺に合わせる）

### 推奨構成
**OP (オープニング): PASTORフォーミュラ**
1. 挨拶・動画概要
2. タイトル回収
3. Problem (悩みの代弁)
4. Amplify (問題の拡大)
5. Solution (解決策)
6. Transformation (変革と証明)
7. Offer Response (初期誘導)

**本編: AREA法**
1. Assertion (衝撃の結論)
2. Reason (根拠ある理由)
3. Example (具体例)
4. Assertion (再度結論)

**ED (エンディング)**
1. エモいメッセージ
2. 公式LINE/登録誘導
3. 復習
4. 挨拶

### 改行ルール
息継ぎや間の取り方がわかるように細めに改行する

### 絵文字使用ルール
適度に使用するが、台本上は間の表現やトーンの補足として用いる

---

## 2. プラットフォーム固有の設定

### 動画の前提条件
台本作成時に必ず以下の情報を確認する
- **テーマの確認**: タイトル・サムネイル案
- **現状の課題**: ターゲットが抱えている具体的な悩み
- **解決策**: この動画で提示する独自の解決策
- **ゴール**: 視聴者が得られる具体的なベネフィット

### SEO・CTA設定
- **タイトル**: 30〜50文字、キーワードを含み、数字やパワーワードを入れる
- **説明文**: 概要、タイムスタンプ、関連リンクを含める

---

## 3. CTA(行動喚起)パターン

### 優先順位
1. 公式LINE登録誘導 / オファー
2. チャンネル登録 / 高評価

### 具体例
「この動画が少しでも役に立ったと思ったら、今のうちにチャンネル登録と高評価をお願いします！そして、概要欄の公式LINEからは…」

---

## 4. AI生成時の注意事項

### 必ず守ること
- ビジネス上流設計(`.agents/rules/mybusiness.md`)のトーン&マナーを遵守すること
- YouTube台本固有の制限やルールを守ること
- **YouTubeの場合**: 必ずPASTORフォーミュラ(OP)とAREA法(本編)の構成を維持し、視聴者の共感と論理的納得を両立させること
- **話し言葉**で、息継ぎや間が伝わるように作成すること
- 重要な箇所は【強調】や「」で目立たせること
- 視聴者の痛みに寄り添い、希望を見せる感情移入表現を心がけること

### 推奨すること
- 視聴者への問いかけを適宜入れ、エンゲージメントを高める
- ベネフィットを具体的に提示する
- 自身の体験やストーリーを交えて信頼構築を行う

### 避けること
- 専門用語の多用(ターゲットが理解できない言葉)
- 根性論や具体性のないアドバイス
- 冗長な導入や結論の見えない構成
