# Antigravity 転職プロジェクト：選考の鉄則 (Rules)

## 1. 自己紹介のマスターテンプレート（30〜60秒）
[cite_start]自己紹介は「面接本編への導入」であり、1分以内で簡潔にまとめ、早く面接官にボールを返すことを最優先する [cite: 47, 48][cite_start]。学歴や詳細な実績は深掘りされた時に話すため、ここでは触れない [cite: 49, 50]。

### 【構成テンプレート】
1. [cite_start]**挨拶と氏名**: 「本日はお時間をいただきありがとうございます。〇〇（氏名）と申します。」 [cite: 46]
2. [cite_start]**職務要約**: 「現職のJAグループ旅行会社では約20年間、法人営業および管理職に従事してまいりました。」 [cite: 18]
3. [cite_start]**強みの提示（Skillsから抽出）**: 「私の強みは『現場の負をITで解決する実行力』です。コロナ禍でリソースが激減する中、独学で習得したGAS等を用い、5名分の業務を1名で完結させる仕組みを構築するなど、生産性向上に寄与してきました。」 [cite: 18, 20]
4. [cite_start]**結び（マッチングの示唆）**: 「本日は、この『営業現場の理解 × デジタル活用』という経験が、御社の〇〇という課題にどう貢献できるかお話しできればと思います。よろしくお願いいたします。」 [cite: 110, 751]

## 2. 評価の二大柱：継続性と再現性
[cite_start]面接は「仕事ができるか（再現性）」と「すぐ辞めないか（継続性）」の2点を証明する場である [cite: 31, 35, 171]。
- [cite_start]**再現性**: 43歳の即戦力として、過去の思考プロセス（STAR法）を用いて「御社でも同様に活躍できる」と確信させる [cite: 95, 103]。
- [cite_start]**継続性**: 「なぜ20年のキャリアを変えるのか？」に対し、不満ではなく「長期的な成長と時代の変化への適応」という軸で答え、定着を証明する [cite: 53, 58]。

## 3. 志望動機：業務内容軸の5段構成
[cite_start]「御社のファン」ではなく「業務への貢献者」として振る舞う [cite: 70, 78]。
1. [cite_start]**結論**: 業務内容が自身のスキル・志向にマッチしていることを伝える [cite: 166, 197]。
2. [cite_start]**前提**: 募集ポジションのリアルな実態（大変さ）を理解していることを示す [cite: 166, 188]。
3. [cite_start]**本題**: その実態に対し、自身のSkills（没頭・貢献・必要）がどう噛み合うかを語る [cite: 167, 208]。
4. [cite_start]**補足**: 社風や理念への共感はサブ要素として短く添える [cite: 167, 211]。
5. [cite_start]**まとめ**: 貢献意欲で締めくくる [cite: 167, 239]。

## 4. 転職理由：ポジティブ・トランスフォーメーション
[cite_start]ネガティブな事実を以下の3要素でフィルタリングして伝える [cite: 55, 412]。
- [cite_start]**長期目線**: 短期的な不満ではなく、5〜10年後の市場価値や成長への懸念とする [cite: 58, 60]。
- [cite_start]**客観的**: 主観的な「辛い」ではなく、残業時間やPC環境のロスなど数字で示す [cite: 60, 62]。
- [cite_start]**試行錯誤**: 「環境が悪いから辞める」のではなく、「自力でDXコミュニティを立ち上げる等の改善を尽くしたが、解消できない構造的課題がある」と伝える [cite: 63, 66]。

## 5. ミドル層・管理職の視座（曽山流）
- [cite_start]**決断経験**: 正解のない状況下（コロナ禍の副業や新規領域挑戦）での「自ら決めた経験」を言語化する [cite: 623, 629]。
- [cite_start]**組織貢献**: 「自分のやりたいこと」を主語にせず、「組織の課題解決のために何ができるか」を主語にする [cite: 634, 638]。
- [cite_start]**弱みの昇華**: 弱みは「ITの仕組み（自動化、デジタルリマインド等）」で120%カバーしている自己管理能力として提示する [cite: 9, 10, 119]。

## 6. Web面接の鉄則
- [cite_start]**非言語情報**: 感情表現を「3割増し」にし、画面越しでも熱意と人間性を伝える [cite: 144, 146]。
- [cite_start]**環境設定**: 目線が下がらない位置にカンペ（Rules/Skillsの要点）を配置し、プロ意識のある背景・照明を整える [cite: 139, 142]。
